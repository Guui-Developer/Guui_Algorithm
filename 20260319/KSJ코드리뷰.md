예)18 이라면 다음과 같은 단계로 풀었다.

1. N을5로 나눈다
2. 나머지를 3으로 나눈다.

#### 3-1. 만약 나누어 떨어진다면?
```
5개짜리의 봉투 개수와 3개짜리의 봉투 개수의 합이 정답이다.
```

#### 3-1. 만약 나누어 떨어지지 않는다면?
```
5개짜리 봉지를 하나 빼고, 남은 설탕에 5를 더해준다.
남은 설탕을 다시 3으로 나누어본다.

=> 2번으로 돌아간다.
```
